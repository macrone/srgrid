Features

Features

sR Grid has twelve features that can be activated as needed:

  1. Wrapper Feature
  2. Hide Feature
  3. Fluid Feature
  4. Offset Feature
  5. Distribution Feature
  6. Reverse Feature
  7. Reorder Feature
  8. Alignment Feature
  9. Align Column Feature
  10. Block Feature
  11. Gapless Feature
  12. Column Fractions Feature

Each feature except the wrapper feature has four different modes.
